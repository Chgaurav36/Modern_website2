# Coffee With Us - Modern Coffee Website

A modern, interactive coffee website featuring smooth animations, video backgrounds, and responsive design.

## Features

- Interactive video background with overlay
- Magnetic hover effects on elements
- Smooth scrolling animations with GSAP
- Responsive design for mobile and desktop
- Image effects with Shery.js
- SEO optimized with meta tags

## Live Demo

[View Live Site](https://modern-website2.vercel.app/)

## Local Development

1. Clone the repository
2. Open `index.html` in your browser, or
3. Run a local server:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- GSAP for animations
- Shery.js for effects
- Three.js for 3D effects

## Assets

- Images in `/images/`
- Videos in `/videos/`
- Fonts in `/font/`

## Deployment

This site is optimized for static hosting platforms like Vercel, Netlify, or GitHub Pages.
